# E-Commerce Sales Analysis

## Overview
This project analyzes e-commerce sales data using Python, NumPy, Pandas, and Matplotlib.

## Features
- Sales Summary
- Product Analysis
- City Analysis
- Category Analysis
- Monthly Sales Analysis
- Data Visualization Dashboard

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
